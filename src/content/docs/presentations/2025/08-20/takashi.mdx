## 概要

Layermate は、Figma のプラグインとして機能する、生成 AI を用いた UI デザイン支援ツールです。自然言語でのチャットによって、ウェブサイトやアプリの UI を自動生成でき、Figma 上で直接操作・編集が可能です。

## 主な特徴

- 自然言語でのデザイン生成
  - 「こういうページが欲しい」とチャットで指示するだけで、UI を自動で生成。見た目だけでなく、レイヤー構造や Auto Layout の設定も整った状態で出力されます。
- コンテキストを保った UI 展開
  - 最初の画面から次の画面へと指示を続けると、前のデザインの「文脈」や「内容」を踏まえた上で画面を生成してくれます。「詳細画面」や「新規作成画面」なども自然につながる構成になります。
- 選択したレイヤーを参照
  - Figma 上の既存のコンポーネントやレイヤーを指定（Attach Selected Layer 機能）することで、それに似せた UI を生成。コンポーネントを参照した再現ではなく、見た目を似せる生成というスタイルです。
- 高速な試作・プロトタイピング
  - 数秒〜数分で UI デザインを生成できるため、アイデア出しや試作フェーズでの活用に特に有効です。エンジニアやデザイン初心者にも扱いやすい設計です。

## プラン

| プラン名   | 価格             | 内容                                                  |
| ---------- | ---------------- | ----------------------------------------------------- |
| Free       | 無料             | 1 日あたり 5 メッセージ／レスポンス長制限あり         |
| Pro        | \$20/月          | 月間 1,000,000 トークン使用可能／長めのレスポンス対応 |
| Boost Pack | \$10（1 回きり） | さらに 50 万トークンを 2 ヶ月間利用可能               |

## 使用した感想

- プロトタイプを高速に作成できるため、顧客への提案の場で大きな力を発揮します
- Free プランは制限が厳しいため、実用的に使うには有料プランへの課金がほぼ必須です
- 学習コストが低く、日本語にも対応しているため、幅広い職種のユーザーにとって扱いやすいツールです
